## 実装した機能

### 1. SiteSettingsモデルの作成
```python
# portfolio/models.py
class SiteSettings(models.Model):
    """サイト全体の設定を管理するモデル"""
    site_name = models.CharField(max_length=200, default='Miyakawa Code', verbose_name='サイト名')
    site_description = models.CharField(max_length=500, blank=True, verbose_name='サイト説明')
    site_keywords = models.CharField(max_length=500, blank=True, verbose_name='SEOキーワード')
    footer_text = models.CharField(max_length=200, blank=True, verbose_name='フッターテキスト')
```

**特徴：**
- サイト設定は1つのみ存在するように制御
- 管理画面から簡単に編集可能
- ハードコーディングを避けた柔軟な設計

### 3. 管理画面の設定
```python
# portfolio/admin.py
@admin.register(SiteSettings)
class SiteSettingsAdmin(admin.ModelAdmin):
    list_display = ['site_name', 'site_description']
    has_add_permission = lambda self, request: not SiteSettings.objects.exists()
    has_delete_permission = lambda self, request, obj=None: False
```
