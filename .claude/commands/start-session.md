---

以下の手順で作業提案を行ってください：

1. CLAUDE.mdを読み込んで、プロジェクトの現在のフェーズを確認
2. reports/フォルダ内の最新のレポート（直近3日分）を確認
3. 前回の「次回やること」を確認
4. 現在のGitブランチとコミット履歴を確認
5. 以下の形式で今日の作業を提案：

## 今日の推奨作業

### 前回までの進捗
[最新レポートから要約]

### 今日やるべきこと
1. [優先度高]
2. [優先度中]
3. [優先度低]

### 確認事項
- 前回の積み残しタスク
- 未解決の問題

提案後、作業を開始するか確認してください。
